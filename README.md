## 🤖 介绍

AI 驱动的 QQ 群不活跃成员检测插件。自动追踪群成员发言活跃度，对长期潜水成员发出警告，超时未响应则自动踢出。支持 AI 拟人化警告、求生申诉、图文周报、活跃排行榜、时段热力图、打卡签到、AI 入群欢迎等功能。

## 📦 安装

- 可以直接在 AstrBot 的插件市场搜索 `astrbot_plugin_group_activity`，点击安装，耐心等待安装完成即可
- 若是安装失败，可以尝试直接克隆源码：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/cunge98/astrbot_plugin_group_activity

# 控制台重启 AstrBot
```

## 📌 使用前须知

- 仅支持 **QQ 平台**（aiocqhttp / OneBot v11）
- 需要 AstrBot 已配置**文转图服务**（Playwright）
- AI 功能需要已配置至少一个 **LLM 提供商**
- 机器人需要**群管理员权限**才能踢人
- 首次使用建议发送 `/初始化活跃数据`

## 🎮 使用说明

### 命令表

| 命令 | 权限 | 说明 |
|------|------|------|
| `/活跃排行` | 所有人 | 查看群活跃排行榜（含 QQ 头像、连续活跃天数） |
| `/活跃查询` | 所有人 | 查询自己的活跃信息和安全状态 |
| `/活跃趋势` | 所有人 | 近 14 天群活跃趋势图（SVG 折线柱状图） |
| `/活跃统计` | 所有人 | 群整体数据摘要（圆环图 + Top3） |
| `/活跃热力图` | 所有人 | 近 14 天发言时段分布热力图（24 小时） |
| `/打卡榜` | 所有人 | 今日打卡排行（首次发言顺序 + 连续活跃称号） |
| `/活跃帮助` | 所有人 | 显示帮助页面 |
| `/活跃检测` | 管理员 | 查看运行状态和配置 |
| `/不活跃列表` | 管理员 | 查看不活跃成员列表 |
| `/手动检测` | 管理员 | 立即执行一次检测 |
| `/初始化活跃数据` | 管理员 | 从平台拉取群成员数据 |
| `/清除警告 [QQ号]` | 管理员 | 清除指定成员的警告状态 |
| `/群周报` | 管理员 | AI 生成图文周报 |
| `@Bot + 理由` | 被警告者 | 向 AI 申诉求生 |

## ⚙️ 配置说明

所有配置项均可在 AstrBot WebUI → 插件管理 → 群活跃检测 中修改：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 全局开关 | 是否启用活跃检测 | 关闭 |
| 图片主题 | 清新蓝 / 活力橙 / 优雅紫 / 暗夜模式 | 清新蓝 |
| 白名单群号 | 仅检测指定群，留空全局检测 | 空 |
| 黑名单群号 | 永不检测的群 | 空 |
| 不活跃天数 | 超过该天数未发言则警告 | 7 天 |
| 警告后踢出 | 警告后多久未响应则踢出 | 24 小时 |
| 检测间隔 | 定时检测频率 | **1 分钟** |
| AI 增强 | 启用 AI 拟人化文案 | 关闭 |
| AI 人设 | 傲娇萌妹 / 严苛群管 / 毒舌段子手 / 古风仙人 / 热血解说员 | 傲娇萌妹 |
| AI 申诉 | 被警告者可 @Bot 申诉 | 关闭 |
| 自动周报 | 每周定时发送 AI 周报 | 关闭 |
| **AI 入群欢迎** | 新成员首次发言时自动发送欢迎语 + 破冰问题 | 关闭 |
| **欢迎语风格** | AI生成 / 活力热血 / 古风雅致 / 简洁清爽 / 自定义 | AI生成 |
| **自定义欢迎语** | 风格选「自定义」时生效，支持 `{nickname}` 变量 | — |

## 📝 更新日志

### v2.2.0
- **活跃时段热力图**：`/活跃热力图` 显示近 14 天 24 小时发言分布，冷暖渐变色直观展示高峰时段
- **活跃打卡/签到**：`/打卡榜` 展示今日打卡排行，按首次发言顺序排列，附连续活跃天数称号
  - 称号体系：🌱 新萌 → 🌿 活跃中 → 🥈 银牌常客 → 🥇 金牌驻场 → 👑 传奇守护者
  - 连续活跃 7 / 14 / 30 天自动在群内发送里程碑贺电
- **AI 入群欢迎**：新成员首次发言时自动引用回复，5 种风格可在 WebUI 下拉选择
  - AI 生成模式：读取新成员的第一句话 + 群名，让 AI 针对性接话/调侃，更幽默自然
  - 重新入群检测：被踢后重新加入的成员也会触发欢迎（下次检测后生效）
- **图表升级**：近 7 天/14 天趋势图改为 SVG 折线柱状图，数值字体加粗+白色描边，一眼看清数据起伏
- **周报修复**：离群成员不再出现在周报排行中
- 检测间隔默认值从 60 分钟改为 **1 分钟**，响应更及时
